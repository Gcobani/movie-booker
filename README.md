## how to install

Download and install [docker](https://docs.docker.com/get-docker/)
Download and install [make](https://www.gnu.org/software/make/)

navigate to the root of the project using your preferred commandline and run `make first`
to install dependencies and to get the project running. In your browser navigate to `http://localhost:8000`
to access the project.
